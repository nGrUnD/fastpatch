## fastpatch v0.2.0

Обновление автозапуска и автоподключения для Windows.

### Установка

Скачайте **`fastpatch_0.2.0_x64_en-US.msi`** (рекомендуется) или **`fastpatch_0.2.0_x64-setup.exe`**.

После обновления с 0.1: в **Настройки → Приложение** выключите и снова включите **Автозапуск**, один раз подключите стратегию вручную (чтобы запомнилась для автоподключения).

### Новое в 0.2

- **Автозапуск через планировщик Windows** — высокий приоритет при входе (раньше типичного старта Discord), без UAC при каждой загрузке ПК (один раз при включении автозапуска в настройках)
- **Автоподключение** последней использованной стратегии после автозапуска (настройка в приложении, по умолчанию включена)
- Надёжное сохранение выбора стратегии в `%USERPROFILE%\AppData\Roaming\fastpatch\prefs.json`
- Повторные попытки подключения при старте с понятной ошибкой в интерфейсе

### Возможности (как в 0.1)

- Подключение одной кнопкой, стратегии с тегами, автоскан, свои `.bat`
- Проверка доступности (порог «медленно» — 3000 ms)
- Скрытый `winws`, трей, настройки zapret и обновлений

### Требования

- Windows 10/11 x64
- Права администратора для winws (UAC при первом ручном запуске или при включении автозапуска)
