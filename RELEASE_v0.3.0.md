## fastpatch v0.3.0

Крупное обновление: **Zapret 2** (winws2 + пресеты), пресет **Apex Legends**, переработанные настройки и автоскан с прогрессом.

### Новое

- Поддержка **Zapret 2** (`winws2.exe`, `.txt` пресеты) и переключение backend v1 / v2
- Пресет **Apex Legends** для Zapret 2: Discord/YouTube + обход EA web; игровой UDP и anti-cheat без лишнего DPI
- Вкладки настроек: **Подключение**, **Стратегии**, **Игры**, **Дополнительно**
- Улучшенные теги стратегий (`recommended`, `aggressive`, `legacy`, `games` и др.)
- **Автоскан** с прогрессом: `N / total`, текущий пресет, прошло / осталось, progress bar
- Безопасное обновление Zapret 2 (staging + откат)
- Панель Apex: пошаговый сценарий, статус пресета и EA-проверки

### Исправлено

- Apex: Discord в пресете, чёрный экран / зависание лобби (исключения EAC, Respawn, AWS для EA App)
- Установка пресета Apex в MSI/NSIS (встроенные ресурсы + `zapret2-extra`)
- Автоподключение при автозапуске только если сохранённая стратегия есть в текущем backend

### Установка

`fastpatch_0.3.0_x64_en-US.msi` (рекомендуется) или `fastpatch_0.3.0_x64-setup.exe`

После обновления: **Настройки → Подключение** — выберите Zapret 2 при необходимости; для Apex — **Игры → Apex Legends**.
